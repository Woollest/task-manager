# 自己学習用レポジトリ

ここでは学習用に作成したタスクを実行した結果をログとして残している。

## 機能特徴

### 成果物1
sample.txtによるコミットの確認

### 成果物2
study_log.shによる学習ログの作成

### 成果物3
study_log.pyによる学習ログの作成

### 成果物4
menu_items.py script.pyによるレストラン注文システムの作成（クラスの学習）

## 開発環境 / 使用技術

- フロントエンド: Python3
- バックエンド: Node.js
- データベース: PostgreSQL
