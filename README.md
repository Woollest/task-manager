# 自己学習用レポジトリ

ここでは学習用に作成したタスクを実行した結果をログとして残している。

## 機能特徴

sample.txtによるコミットの確認

## 開発環境 / 使用技術

- フロントエンド: Python3
- バックエンド: Node.js
- データベース: PostgreSQL
